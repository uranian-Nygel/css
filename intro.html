css - cascading style sheets

CSS is a programing language used to improve the presentation of a html page

CSS will help modify aspects like text size , color , background , image size and shapes , borders,poistions responsiveness (different looks on different screen sizes) among other aspects

CSS is used to design user interfaces for web pages and it helps enhance user experience.

CSS syntax(how do we write css)

A selector is used to target an element or group of elements for styling

{} - A declaration block - group properties and value to be applied on the selector

Property- a presentation aspect you'd want to modify e.g color, width, size, position.

Value - final look you are interested in for the page(red, fullpage,400px...)

        
            selector{
                property: value;
            }
        
        
Example

        
            h1{
                color: red;
            }
